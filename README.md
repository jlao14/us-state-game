# us-state-game
An interactive, educational Python game that tests your geographical knowledge of the United States. Built using Python's `turtle` graphics module and `pandas` for data manipulation.

## Features
- **Interactive Visual Map:** Correct guesses automatically populate on a blank map of the U.S. at their exact coordinates.
  
- **Score Tracking:** Keeps a running tally of your correct guesses (out of 50).
  
- **Study Aid Generation:** If you decide to exit early by typing "Exit", the program automatically generates a `states_to_learn.csv` file containing a list of all the states you missed.

## Prerequisites
To run this project, you will need Python installed on your system along with the `pandas` library.

## Setup & Installation
1. Clone this repository or download the project files.
2. Install the required `pandas` library by running the following command in your terminal:

   pip install pandas
